# Digital-literacy-project

This project is part of the CSE0001 Digital Literacy course. It focuses on developing digital awareness, practical skills, and responsible use of technology.

---

## 📌 Tasks Included

### 🔹 Task 1: Infographic
Created a digital literacy infographic using Canva. It includes safe internet practices, digital tools, and responsible online behavior.

### 🔹 Task 2: Digital Portfolio
Built a professional digital portfolio using:
- GitHub (projects showcase)
- LinkedIn (professional profile)
- Kaggle (data science practice)

### 🔹 Task 3: Quiz & Coding
- Solved a coding problem on HackerRank  
- Created a Google Form quiz with MCQs, correct answers, and marks

### 🔹 Task 4: Email Etiquette
Learned and implemented professional email writing with proper structure and tone.

- Created a prevention checklist
- Wrote a personal reflection
